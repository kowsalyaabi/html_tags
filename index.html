List

  • In HTML, a list is a way to organize and present related information. There are three main types:
    • Ordered List (<ol>):
    • Items are numbered sequentially.
      • See Example
    • Unordered List (<ul>):
    • Items are bulleted.
      • See Example
    • Description List (<dl>):
    • Terms and their definitions.
      • See Example

Ordered List Example

  1. apple
  2. orange
  3. banana

Unordered List Example

  • apple
  • orange
  • banana

Description List Example

Fruits
apple
orange
Vegetables
Tomato
Potato
